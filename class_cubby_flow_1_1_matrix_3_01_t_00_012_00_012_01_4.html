Detailed Description

template<typename T>
class CubbyFlow::Matrix< T, 2, 2 >

2-D matrix class.

This class is a row-major 2-D matrix class, which means each element of the matrix is stored in order of (0, 0), (0, 1), (1, 0) and (1, 1).

Template Parameters
T- Type of the element.
