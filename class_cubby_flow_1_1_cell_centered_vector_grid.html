Detailed Description

template<size_t N>
class CubbyFlow::CellCenteredVectorGrid< N >

N-D Cell-centered vector grid structure.

This class represents N-D cell-centered vector grid which extends CollocatedVectorGrid. As its name suggests, the class defines the data point at the center of a grid cell. Thus, the dimension of data points are equal to the dimension of the cells.
